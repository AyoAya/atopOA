四川华拓光通信股份有限公司是中国（绵阳）科技城金家林总部经济试验区的一家国有注资的股份公司，注册资本2670万元，在四川绵阳自有总占地30000平米的工业园区--华拓光通信信息港。

华拓光通信是一家行业领先的集研发、生产、销售和服务为一体的光模块制造商。公司的业务领域涉及光器件、模块化产品直至子系统的研发、生产和销售，公司目前拥有155M-100G光收发模块、高速互连产品40G QSFP+ AOC/DAC Cable、10G SFP+ AOC/DAC Cable，为云计算和大数据中心、数据通信、存储网络、长途传输、无线接入等领域客户提供最佳的光通信模块解决方案。
