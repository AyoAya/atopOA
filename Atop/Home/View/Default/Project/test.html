项目描述：

“本项目结 构上分为表现层、业务层和数据访问层，层次间的依赖关系自下到上。 采用的技术有Struts，Spring，Hibernate，Log4J，JDom 等。其中表现层采 用Struts框架开发；业务层封装业务流程，为适应业务的变更，每一业务模块均 有专门的接口及实现类，利用Spring的IoC功能将 实现类注入给表现层的Action； 数据访问层借助于Hibernate实现，代码简洁且可适应不同的数据库。事务部分利 用Spring的声明式事务管 理。为提高性能，采用Servlet Filter实现了缓存代理”
